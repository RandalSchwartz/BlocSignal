# Flutter Hook References

This directory contains individual markdown guides for every signal-based hook available in `signals_hooks`.

## Documented Hooks

The files cover all major reactive hook primitives including:
- **`useSignal`** / **`useComputed`** / **`useSignalEffect`**: Basic signals hooks.
- **`useAsyncSignal`** / **`useFutureSignal`** / **`useStreamSignal`**: Asynchronous and stream binding hooks.
- **`useListSignal`** / **`useMapSignal`** / **`useSetSignal`**: Collection-specific reactive hooks.
- **`useConnect`**: Complex multi-signal stream connectors.
- **`useValueNotifierToSignal`**: Standard Flutter state bridging helpers.
